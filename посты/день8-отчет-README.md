## Как использовать

### 1. Сделать скриншот визуала

1. Открой `визуал/деньN-отчет.html` в браузере
2. F12 → режим устройства (Ctrl+Shift+M) → 1080×1350
3. Скриншот (например, через Cmd+Shift+4 на Mac)
4. Сохрани как `день8-отчет.png`

### 2. Опубликовать в Telegram

1. Открой `@benzyn_ryadom`
2. Создай пост
3. Прикрепи скриншот
4. Скопируй текст из `день8-отчет.md`
5. Отправь

### 3. Опубликовать в VK

1. Открой сообщество `vk.com/benzyn_ryadom`
2. Создай пост
3. Прикрепи скриншот
4. Скопируй текст из `день8-отчет-vk.md`
5. Опубликуй
