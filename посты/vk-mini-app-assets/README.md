• Ваши собственные отчёты

⚡ В реальном времени:

• 🔔 Подписка на завоз — пуш когда появится топливо
• 🏆 Репутация и бейджи за активность

💡 Бот бесплатный. Поддержать проект: vk.com/donut/benzyn_ryadom

## 🎬 Lottie loading.json

Двойной спиннер + пульсирующий центр. 3 слоя:
- **Outer Ring** (80×80) — синий, вращение 360°, 1/4 окружности
- **Inner Ring** (56×56) — фиолетовый, вращение -360°, 1/8 окружности  
- **Center Dot** (16×16) — синий, пульсация opacity 60→100%, scale 80→110%

60 FPS, 60 кадров = 1 секунда анимации, зациклено.

## 🚀 Как загрузить

1. **VK → Сообщество → Управление → Настройки → Работа с API → Mini Apps**
2. Откройте приложение (или создайте)
3. Загрузите файлы из таблицы выше
4. Вставьте описания
5. Сохраните
